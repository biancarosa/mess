mess - a messaging system

Arquitetura

O sistema consiste em vários nós com um servidor e múltiplos clientes.

Cliente

Cada cliente fica esperando um input do usuário com uma mensagem e mostra mensagens enviadas pelo servidor, provenientes de outro usuário

A tecnologia utilizada para o envio e recebimento de mensagens do lado do cliente é uma biblioteca chamada Pyro (Python Smart Objects).

O cliente faz um lookup em um servidor de nomes e executa uma função exposta por um objeto no servidor de forma remota.

O servidor embutido em cada client roda em uma thread daemon(filha, não-bloqueante).

Servidor

O servidor recebe mensagens de um cliente, envia para um grupo via socket multicast e recebe as mensagens desse grupo e envia para os clientes.

O servidor que fica ouvindo o socket multicast roda em uma thread daemon(filha, não-bloquante).
